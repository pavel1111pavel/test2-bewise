Описание
Данный проект представляет собой REST-сервис, позволяющий создавать пользователей, добавлять аудиозаписи и получать доступ к аудиозаписям по ссылкам.

Требования
Docker
Docker Compose
Установка и запуск
Склонируйте репозиторий:


git clone https://github.com/your-repo/your-project.git
cd your-project

Убедитесь, что у вас установлены Docker и Docker Compose.

Запустите контейнер с PostgreSQL и веб-сервисом:

docker-compose up -d
Это создаст и запустит контейнеры на основе предоставленных Docker-образов.

Подождите, пока контейнеры будут запущены и настроены.

Проверьте, что контейнеры успешно запущены:

docker-compose ps
Вы должны увидеть список контейнеров, включая контейнер с PostgreSQL и веб-сервисом.

Теперь вы можете использовать REST-методы для взаимодействия с веб-сервисом. Документация API доступна по адресу http://localhost:8000/docs.

Использование

Создание пользователя
Метод: POST
Путь: /users
Тело запроса:

{
  "username": "your_username"
}

Ответ:

{
  "userId": "user_id",
  "accessToken": "access_token"
}


Добавление аудиозаписи

Метод: POST
Путь: /records
Тело запроса:
Формат данных: multipart/form-data
Поля:
userId: уникальный идентификатор пользователя
accessToken: токен доступа
audio: аудиозапись в формате WAV
Ответ:

{
  "recordId": "record_id",
  "downloadUrl": "http://localhost:8000/record?id=record_id&user=user_id"
}

Доступ к аудиозаписи

Метод: GET
Путь: /record
Параметры запроса:
id: идентификатор аудиозаписи
user: идентификатор пользователя
Ответ: аудиозапись в формате MP3
